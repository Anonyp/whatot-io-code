小问题

这是一道简单的几何问题，假设在座标轴上由A点移动到B点，如何计算B相对于A的偏移？这个问题对于我们来说是非常的简单，可能大部分人都会脱口而出并得到答案为B-A。

那么这个答案是否完全准确呢？比较严谨的你觉得显然不是，原因在于，当A为坐标原点即A=0的时候，上述答案B-A就直接简化为B了。
